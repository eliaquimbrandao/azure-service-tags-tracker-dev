Why Teams Rely on This Azure IP Range Tracker

Whether you manage firewalls, craft conditional access policies, or maintain allow-lists, this Azure IP range tracker keeps Microsoft Service Tags changes transparent, explainable, and easy to automate.

Microsoft-Sourced JSON, Updated Weekly

The platform pulls the official Azure Service Tags JSON every Monday at 07:00 UTC via GitHub Actions. Each run produces current.json, differential reports, and timestamped history so you always know when an IP range was added or retired.

Built for Security & Networking Teams

Search for services like AzureStorage or regions such as East US, export change sets, and sync results with firewall automation. Optional email subscriptions summarize Azure IP range movement for stakeholders that prefer inbox alerts.

API & Automation Friendly

Every dataset lives on GitHub Pages under /data. Tools can fetch summary.json to discover available dates, compare "/history/" snapshots for precise diffs, or read manifest.json to queue downloads. Sample scripts for PowerShell, Python, JavaScript, C#, Go, Java, and Ruby are provided in the repository.
